# Recess Autoresearch

Autonomous prompt optimization for Recess, inspired by [Karpathy's AutoResearch](https://github.com/karpathy/autoresearch).

## What Is This?

An automated loop that improves any prompt or agent workflow by:
1. Making one small change
2. Scoring the output against binary eval criteria
3. Keeping improvements, reverting failures
4. Repeating 50-100 rounds overnight

## Current Status

**Planning phase.** The task packet (`task-packet.md`) contains the full architecture, implementation plan, eval criteria, and cost estimates.
